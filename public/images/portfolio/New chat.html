I am writing the website using React. How can I make a React component that has the effect of following the mouse with cool colors.

There are several ways to create a React component that follows the mouse with cool colors. One way is to use the useState hook to store the current mouse position and the useEffect hook to update the position whenever the mouse moves. Here is an example of a basic component that follows the cursor and changes color when hovered:

import React, { useState, useEffect } from 'react';

function FollowCursor() {
  const [x, setX] = useState(0);
  const [y, setY] = useState(0);
  const [color, setColor] = useState('blue');

  useEffect(() => {
    const updateMousePosition = (event) => {
      setX(event.clientX);
      setY(event.clientY);
    }

    document.addEventListener('mousemove', updateMousePosition);

    return () => {
      document.removeEventListener('mousemove', updateMousePosition);
    }
  }, []);

  return (
    <div
      style={{
        position: 'fixed',
        left: x,
        top: y,
        width: '20px',
        height: '20px',
        backgroundColor: color,
        borderRadius: '50%'
      }}
      onMouseEnter={() => setColor('red')}
      onMouseLeave={() => setColor('blue')}
    />
  );
}

export default FollowCursor;

You can customize the component by changing the size and shape of the cursor and also by changing the color logic.

It is also possible to use CSS animations to follow the cursor and change colors, but this approach would not be React specific. Please also keep in mind that having a cursor following the mouse too much can be quite distracting for the user.
